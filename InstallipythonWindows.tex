        \subsection{Installing ipython notebook on Windows}

\begin{enumerate}[1.]
\item
  Download and install
  \href{http://www.enthought.com/downloads/}{Enthought Canopy}, the free
  version. This might take some time. When you open it and it asks you
  to make Canopy the default environment for python, say yes.
\item
  Open \texttt{cmd.exe} by going to the Start menu and searching for
  \texttt{cmd.exe}. This is a version of the terminal for Windows (it
  has slightly different commands than the Linux terminal, but you
  shouldn't need many). It should look like this: 
\item
  Type: \texttt{enpkg enstaller} and press enter.
\item
  Type: \texttt{enpkg ipython} and press enter
\end{enumerate}\subsection{To open ipython notebook}

Now when you want to actually work on \texttt{ipython notebook}, you
need to run it from the cmd prompt, NOT the Enthought Canopy program you
just downloaded.

\begin{enumerate}[1.]
\item
  Open \texttt{cmd.exe}
\item
  In the Windows command prompt, the Linux \texttt{ls} command is called
  \texttt{dir}. \texttt{cd} is the same. You need to \texttt{dir} /
  \texttt{cd} into the folder where you want to save all your ipython
  notebooks in (and save all your images, datasets, etc): 
\item
  In the cmd prompt, to open ipython notebook, type
  \texttt{ipython notebook -{}-pylab inline}. This will open the ipython
  notebook in Firefox. Create a new ipython notebook by clicking ``New
  Notebook''. This is where you do all your work! 
\end{enumerate}
