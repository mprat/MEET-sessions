        \subsection{Installing Git (GitBash) on Windows}

\begin{enumerate}[1.]
\item
  Download and install GitBash from the first link here:
  http://code.google.com/p/msysgit/downloads/list?q=label:Featured
\item
  Choose all the defaults when installing (do not change anything unless
  you REALLY know what you're doing). Just keep pressing Next.
\item
  Open GitBash from the Start Menu. DO NOT USE GIT GUI.

   
\item
  Tell Git about your username and email. Type the following:
\end{enumerate}
\begin{quote}
git config --global user.name ``YOUR NAME''

\end{quote}
\begin{quote}
git config --global user.email ``mprat@meet.mit.edu''

\end{quote}
Instead of \textbf{YOUR NAME} and \textbf{mprat@meet.mit.edu}, write
your own name and your own (personal) email address.\subsection{Using Git on Windows}

When you need to use Git, open GitBash. This is a terminal exactly like
we use on the Ubuntu machines. \texttt{cd} onto the Desktop and
\texttt{git clone} your repository so you have the files. The difference
is that you can't run \texttt{ipython notebook} or \texttt{idle} or
\texttt{python} from GitBash - you need to use the Windows cmd as
described in the other document.
